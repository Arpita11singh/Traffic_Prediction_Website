**Road Sarthi** is an intelligent, full-stack web application designed to take the guesswork out of daily commutes in India. By leveraging the power of **Google's Gemini 2.0 Flash AI**, this platform provides real-time traffic forecasting, optimal departure times, and personalized route alerts. 

Whether you are planning a one-time trip or setting up daily office commute reminders, Road Sarthi acts as your personal smart assistant for the road.

## ✨ Key Features

* **🔮 AI Traffic Forecasting:** Predicts traffic congestion levels and provides commuter advisories for specific Indian cities, dates, and times using the Gemini API.
* **⏱️ "Best Time to Leave" Calculator:** Input your destination and desired arrival time, and the AI will calculate the absolute best time to start your journey.
* **🗺️ Personalized Commute Alerts:** Save your frequent routes (e.g., Home to Office) and set daily alerts. Powered by **Firebase Auth & Firestore** for secure, real-time data synchronization.
* **📊 Historical Data Dashboard:** Visualize traffic trends over time using interactive **Chart.js** graphs.
* **📂 Custom Data Analysis:** Upload your own traffic datasets (Excel/CSV via SheetJS & PapaParse) and let Gemini analyze the trends to extract actionable insights.
* **🤖 Integrated Help Bot:** A built-in AI chatbot ready to answer your traffic and commute-related queries on the fly.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Vanilla JavaScript, Tailwind CSS
* **Backend/BaaS:** Firebase Authentication (Email/Password & Anonymous), Cloud Firestore
* **AI Integration:** Google Gemini 2.0 Flash API
* **Data Visualization & Parsing:** Chart.js, SheetJS (XLSX), PapaParse
